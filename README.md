### PAYCOINS

É uma `aplicação móvel` que ajuda a todos jogadores de jogos electrónicos a poderão comprar moedas de jogos sem necessariamente ter um cartão de crédito, utilizando apenas seu numero 84 ou 85 para fazer compras. .

link de download: https://paycoins.vercel.app

## how to clone

To clone, just have git installed and run the command: git clone https://github.com/DenyCelestino/paycoins.git

## how to install dependencies

Once downloaded, run the command: `npm` or `yarn install`.

And finally run : `expo start`
And enjoy 😊❤️🎉
